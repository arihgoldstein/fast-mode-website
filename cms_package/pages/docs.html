Getting Started

Welcome to FastMode! This documentation will help you get your first website up and running in minutes.

What is FastMode?

FastMode is a modern CMS and website hosting platform that lets you:

  • Deploy websites instantly — Upload your HTML/CSS/JS or connect GitHub for auto-deployments
  • Edit content visually — Update text, images, and content without touching code
  • Manage structured content — Built-in blog, team, and custom collections
  • Connect with 5,000+ apps — Zapier integration for forms and CMS automation
  • Use your own domain — Free SSL certificates included

Your First 5 Minutes

  1. Sign up at app.fastmode.ai
  2. Create a site — Pick a name and subdomain
  3. Deploy — Connect GitHub or upload a ZIP
  4. Edit — Use the visual editor to update content
  5. Go live — Your site is instantly available at yoursite.fastmode.ai

Who is FastMode For?

  • Developers building client sites who want easy handoff for content editing
  • Designers who want to deploy static sites with a built-in CMS
  • Agencies managing multiple client websites
  • Small businesses who need a simple, fast website with easy updates
  • AI builders using tools like Cursor, Claude, or ChatGPT to generate websites
